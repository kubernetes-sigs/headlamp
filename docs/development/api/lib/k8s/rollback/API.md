# lib/k8s/rollback

## Interfaces

| Interface | Description |
| ------ | ------ |
| [RevisionInfo](interfaces/RevisionInfo.md) | Represents a single revision in the history of a rollbackable resource. Used by RevisionHistorySection and RollbackDialog to display revision details. |
| [RollbackOptions](interfaces/RollbackOptions.md) | Options for a rollback operation. |
| [RollbackResult](interfaces/RollbackResult.md) | Result of a rollback operation on a workload resource. |
